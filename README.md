Seek&Destroy RUtrolls accounts on YouTube
--------------------------------------------------------


Setup
------

1. Create env (conda)
2. Create .env file with YouTube API_KEY (google how to get it), API_KEY=<your key>
3. Add ATTEMPTS=100000 variable to .env files -- the number of random variations to try
4. python main.py
5. check logs
